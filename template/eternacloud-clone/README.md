# EternaCloud Website Clone

A faithful clone of the [EternaCloud](https://www.eternacloud.com/) website with all main pages.

## Structure

- **resources/** – All static assets (images, SVGs, fonts, JSON, videos)
- **static/** – All HTML, CSS, and JavaScript files

## Pages (in static/)

- **index.html** – Home (Mission Critical Continuity)
- **what-we-do.html** – Critical Equipment Orchestration
- **our-approach.html** – Eliminate the Hassle
- **about-us.html** – Our Business is Clarity, Consistency & Unity
- **insights.html** – Untapped Opportunity
- **contact-us.html** – Contact

## How to View

1. Open `static/index.html` in a browser, or open root `index.html` (redirects)
2. Or use a local server: `npx serve .` or `python -m http.server 8000`
3. An internet connection is needed for CDN assets (some images, GSAP, Swiper, Lenis)
